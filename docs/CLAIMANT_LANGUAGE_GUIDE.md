# Claimant Language Guide

Use plain claimant-facing language by default.

Preferred wording:

- Local rent area, not BRMA.
- Local housing limit, not LHA.
- Calculation breakdown, not derived artifacts.
- This situation cannot yet be calculated safely, not unsupported artifact.
- Employment payment entered by you, not employment_manual.

Technical wording may appear only in expandable technical details, audit views, developer-only views, or documentation.

Optional first names may personalize the interview. Names are presentation metadata only and must not affect calculation logic.
